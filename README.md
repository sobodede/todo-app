# Todo App

Application de gestion de tâches réalisée en PHP / MySQL / JavaScript.

## Fonctionnalités
- Ajouter une tâche
- Modifier une tâche
- Supprimer une tâche
- Afficher les tâches
- Filtrer les tâches (toutes / à faire / terminées)
- Validation JavaScript (titre obligatoire)

## Technologies utilisées
- PHP
- MySQL
- JavaScript
- HTML
- CSS
